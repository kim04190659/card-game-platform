# カードゲーム基盤ソフトウェア

## 概要

教育・コンサルティング用の汎用カードゲーム基盤システム

## 開発環境

- Node.js 18+
- Vercel CLI

## セットアップ
```bash
# 依存関係のインストール
npm install

# ローカル開発サーバー起動
npm run dev
```

## デプロイ
```bash
# 本番環境にデプロイ
npm run deploy
```
